formir

Formir is pure CSS and HTML framework for developing responsive web design pages.

The idea is to minimize the need for JS code and rely on UI elements on pure html based on clean, easy to use and smart CSS code.

Build your own template based on included mixing, fast, clean and easy. No more !important.
