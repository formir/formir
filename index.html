Hello World!

I'm first pure, complete and modular
CSS & HTML Framework for RWD developing.

Our idea is:

  • to minimize JS code in application - so we don't have any
  • no more !important - we give you free hand with style,
  • use less style as possible,
  • give you able to set every class your own name,
  • propose you to use structure of files gained of our experience,
  • do it all with the help of modules and mixins.
Well, hello there, developer - this is formir
